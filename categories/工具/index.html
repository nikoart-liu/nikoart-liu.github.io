一、环境准备及 Hugo 安装 安装 Homebrew 如果你还没有安装 Homebrew，请先打开终端执行以下命令来安装这个 macOS 上最常用的包管理器： /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" 使用 Homebrew 安装 Hugo Homebrew 安装包的方式非常简洁，一条命令即可完成： brew install hugo 安装完成后，验证 Hugo 是否正确安装： hugo version 如果终端显示类似 hugo v0.155.0+extended+withdeploy darwin/arm64 BuildDate=2026-01-28T16:28:23Z VendorInfo=Homebrew 的版本信息，说明安装成功。 二、创建 Hugo 站点 初始化站点 使用 Hugo 命令创建一个全新的网站基础结构： hugo new site my-blog --format=yaml cd my-blog git init 执行完成后，Hugo 会创建一个名为 my-blog 的目录。进入这个目录查看自动生成的文件结构： my-blog/ ├── archetypes/ │ └── default.md ├── content/ ├── layouts/ ├── static/ ├── themes/ └── hugo.yaml 这个结构的设计非常清晰： ...
